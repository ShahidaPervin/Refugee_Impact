## Creating Bangladesh Map
This map was created on Google colab on December 25, 2025: https://colab.research.google.com <br>
The shape files were collected from from GADM December 25, 2025: https://gadm.org/download_country.html <br>
To reproduce the map: Either, run the program MapProgram.ipynb on Google colab. Shape files are available in Shapefiles folder <br>
Or, directly run BDmap.ipynb on colab where zip files are downloaded from the website
